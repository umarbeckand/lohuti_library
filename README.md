# Библиотека "Лохути"

Веб-приложение для управления библиотекой с возможностью поиска книг.

## Описание

Сайт библиотеки "Лохути" позволяет пользователям просматривать каталог книг и искать нужную литературу по названию, автору или жанру.

## Функционал

- 📚 Просмотр каталога книг
- 🔍 Поиск книг по названию, автору, жанру
- 📖 Просмотр детальной информации о книге
- 📝 Онлайн-запись на получение книги (если есть)

## Технологии

- **Backend:** Python 3.x, Django
- **База данных:** SQLite
- **Frontend:** HTML, CSS,  
- **Шаблонизатор:** Django Templates

## Установка и запуск

### 1. Клонирование репозитория
```bash

cd lohuti-library
```

### 2. Создание виртуального окружения
```bash
python -m venv venv
```

**Windows:**
```bash
venv\Scripts\activate
```

**Linux/Mac:**
```bash
source venv/bin/activate
```

### 3. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 4. Применение миграций
```bash
python manage.py migrate
```

### 5. Создание суперпользователя (опционально)
```bash
python manage.py createsuperuser
```

### 6. Запуск сервера
```bash
python manage.py runserver
```

Сайт будет доступен по адресу: `http://127.0.0.1:8000/`
