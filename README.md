# Lohuti Library Management System

Система управления библиотекой имени Лохути, разработанная на Django.

## 📚 Описание

Веб-приложение для автоматизации работы библиотеки. Проект предназначен для управления книжным фондом, учета читателей и оптимизации библиотечных процессов.

## ⚙️ Технологии

- **Python** - основной язык программирования
- **Django** - веб-фреймворк
- **SQLite** - база данных
- **HTML/CSS/JavaScript** - фронтенд

## 🚀 Установка и запуск

### Требования
- Python 3.8 или выше
- pip (менеджер пакетов Python)

### Шаги установки

1. Клонируйте репозиторий:
```bash
git clone https://github.com/umarbeckand/lohuti_library.git
cd lohuti_library
```

2. Создайте виртуальное окружение:
```bash
python -m venv venv
```

3. Активируйте виртуальное окружение:

**Windows:**
```bash
venv\Scripts\activate
```

**Linux/Mac:**
```bash
source venv/bin/activate
```

4. Установите зависимости:
```bash
pip install -r requirements.txt
```

5. Примените миграции:
```bash
python manage.py migrate
```

6. Запустите сервер разработки:
```bash
python manage.py runserver
```

7. Откройте браузер и перейдите по адресу:
```
http://127.0.0.1:8000/
```
